BTCUSD ⟶ replaces BOOM500 as Phase-A instrument #3

Implementation plan · 2026-06-17 · LSB Strategy Bot (Phase A, Session A3b)

1 Overview

BOOM500's only history source (Deriv ticks_history) caps at ~1 year, but the walk-forward window needs 24 months (18 train + 6 test) — so BOOM500 cannot complete an end-to-end walk-forward. We substitute BTCUSD as the third instrument: it is sourced from the same Dukascopy fetcher already proven for EURUSD/XAUUSD (multi-year H1 depth, zero new data-source code), and preserves the third-asset-class diversity the multi-instrument stability test needs — 24/7, fat-tailed, no weekend gaps, the closest "spirit" match to a Boom synthetic.

GOVERNANCE The instrument list is Appendix A, inherited unchanged from Blueprint v2.1, which is immutable during Phase A. This substitution is therefore a governed decision recorded as ADR-002, and it is approved by the user for this session.
